Detection.maskrcnn_benchmark.modeling.backbone.resnet module

Variant of the resnet module that takes cfg as an argument. Example usage. Strings may be specified in the config file.

model = ResNet(

“StemWithFixedBatchNorm”, “BottleneckWithFixedBatchNorm”, “ResNet50StagesTo4”,

)

Custom implementations may be written in user code and hooked in via the register_* functions.
