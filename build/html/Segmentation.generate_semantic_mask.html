Segmentation.generate_semantic_mask module

class Segmentation.generate_semantic_mask.Segmentation(network_path)[source]

Bases: object

This class is used for segmentation of the frame into 7 classes [“background”, “race_track”, “jockey_horse”, “foreground”, “pole”, “rail”, “static”] race_track refers to the grass track jockey_horse refers to the combination of jockey bodies and horses pole refers to the light poles in the field static refers to the additional display overlay on the frame by the tv broadcast
