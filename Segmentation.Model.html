Class for using DeepLab Model

Parameters

  • network_path (string) – path for the network to restore

  • channels (int) – input channels, default is 3

  • n_class (int) – number of output class

  • img_row (int) – width of input image

  • img_cols (int) – height of input image

  • cropping (int) – the amount of cropping at test, optimum is 35
